## Getting Started 🚀

1.  Clone the repository:
    ```bash
    git clone https://github.com/yamneg96/JS-Projects
    ```
2.  Navigate to the project directory you want to explore:
    ```bash
    cd <project_directory>
    ```
3.  Open the `index.html` file in your web browser.
